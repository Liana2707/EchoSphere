# Модель динамики мнений с учётом алгоритмического влияния в социальных сетях
## 📌 Описание проекта

Данный проект посвящён разработке микроуровневой модели динамики мнений пользователей в социальной сети с учётом влияния алгоритмов рекомендаций, распределения внимания и когнитивных механизмов усвоения информации.

Цель работы — количественно описать, как меняются мнения отдельных пользователей под воздействием алгоритмически отобранного контента, и при каких условиях система переходит в предельное состояние эхо-камеры.

Проект находится на стыке:

анализа социальных сетей,

вычислительной социологии,

моделирования сложных систем,

рекомендательных систем.
